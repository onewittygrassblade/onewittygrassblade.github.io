Pixie Perilousness!

A short side scroller for the browser built with Pixi.js.

Click here to play

Source code on Github

After completing Rex van der Spuy's excellent tutorial on Pixi.js, I decided to expand the final project, a fairly minimalistic side scroller, into a fully playable short little game. Notable added features include:

  • Game states: various screens (Title, About, How To Play, Game Over), ability to pause the game.
  • Pick ups for points or random effects (extra life, invicincibility...).
  • Four levels with different game mechanics and increasing difficulty.
  • Score rating based on pick ups.
  • Music and sound effects using PixiSound.
  • Use of Webpack to structure the project into several files and folders and output a minified bundle.

This game is mainly controlled via keyboard and is not suitable for mobile devices.
